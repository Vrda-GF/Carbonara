Carbonara is a mathematics software which makes it easy for users to analyse, explore, visualise and simulate various problems regarding optimization, numerical and regression analysis. In this Variant on each Turn, a Player and the PC Computer must remove/take any Number of Coins' Stack from one Row alternately. The one, who takes the last Stack of Coins loses.<br />
